Desenvolvedor .NET - Fullstack

Sou um desenvolvedor de software com foco em Backend utilizando tecnologias como C# e .NET, atuando com ASP.NET Core na construção de APIs REST escaláveis e seguras. Tenho experiência com SQL Server e integração com o Entity Framework para gerenciamento eficiente de dados. Trabalho com Visual Studio como IDE principal, aproveitando seus recursos avançados para produtividade e depuração.

Trabalhei com metodologias ágeis, utilizando ferramentas como o Trello, e aplicando o Scrum como metodologia ágil.

Também possuo experiência com TypeScript em cenários complementares ao backend, especialmente na integração e consumo de APIs REST, contribuindo para a manutenção e evolução de aplicações full stack com foco na consistência e escalabilidade.
